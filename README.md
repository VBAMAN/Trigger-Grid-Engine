## 🚀 Concept

This engine separates:

- Data (cell values like "RD", "GR")
- Logic (triggers)
- Structure (areas)
- Presentation (palette)

Everything is reconstructed from saved data.

---

## 🧠 Key Ideas

- Trigger-driven execution
- Data-driven architecture
- Editor mode / Execution mode separation
- Full reconstruction from save data
- Palette-based rendering (code → color)

---

## 🧩 Workflow

1. Build structure (Editor mode)
2. Save data
3. Load in execution mode
4. Apply palette
5. Run triggers
